How To Play

Explore by dragging around the map. The game speaks what you touch.

Double tap a building, person, room door, or object to interact immediately. There is no Activate button.

Every building and room uses the same touch exploration system.

Events should tell you who, where, impact and follow-up. If something breaks, it becomes a tracked issue.

Sound files should sit beside this HTML file. Missing sound files do not stop the game working.
